
# E-872 controller samples

## General notice

Please refer to the general readme file located here: ..\..\ReadMe.txt.
It describes general topics about how to work with the samples.

## Provided samples for this controller type

### MoveSample
    This sample connects to a E-872 controller using a connection dialog. It
    shows how to execute an open-loop motion on the connected channels. The
    controller and positioners have to be initialized with PIMikroMove before
    starting this sample.
