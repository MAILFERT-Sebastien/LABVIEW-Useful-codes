
# C-891 controller samples

## General notice

Please refer to the general readme file located here: ..\..\ReadMe.txt.
It describes general topics about how to work with the samples.

## Provided samples for this controller type

### DataRecorderSample
    This sample connects to a C-891 controller, starts up the connected
    positioner and references it. Then the data recorder is configured and
    samples of absolute and relative moves are captured. They are written
    to std::cout and also to a file.

### ReferenceSample
    This sample connects to a C-891 controller, starts up the connected
    positioner and references it.
